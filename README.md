# Global Retail Sales Analysis using SQL Server

##  Project Overview

This project analyzes global retail sales data using SQL Server to generate meaningful business insights. The project demonstrates SQL skills through data validation, data modeling, joins, window functions, aggregate functions, date functions, and business scenario-based queries.

## 📊 Project Objectives

- Analyze global retail sales performance
- Generate business insights using SQL queries
- Practice real-world business scenarios
- Improve SQL problem-solving skills

## 🛠️ Technologies Used

- SQL Server
- SQL Server Management Studio (SSMS)

## 📂 Dataset

The project contains the following datasets:

- Customers
- Products
- Sales
- Stores
- Exchange Rates
- Data Dictionary
